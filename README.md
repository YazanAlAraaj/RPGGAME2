# ⚔️ My Game — A C# RPG

A classic text-driven **role-playing game** built with C# and Windows Forms. Explore locations, fight enemies, complete quests, and level up your character!

## 🎮 Features

- 🗺️ **Exploration** — Move between locations and discover what lurks within
- ⚔️ **Turn-based Combat** — Fight enemies like rats, track damage dealt and received
- 🎒 **Inventory System** — Collect and use items such as weapons and potions
- 📜 **Quest Log** — Accept quests and track your progress
- 📈 **Character Progression** — Gain experience, gold, and level up
- 📝 **Event Log** — A scrollable log of everything that happens in the world

---

## 🧙 Gameplay

You start with basic stats and a **Rusty Sword**. Navigate the world using directional buttons, engage enemies in combat, and complete quests to earn rewards.

**Example quest:**
> *"Kill rats in the alchemist's garden and bring back 3 rat tails. You will receive a healing potion and 10 gold pieces."*

---

## 📊 Character Stats

| Stat        | Description                        |
|-------------|------------------------------------|
| Hit Points  | Your current health                |
| Gold        | Currency used for trades           |
| Experience  | Earned by defeating enemies        |
| Level       | Increases as you gain experience   |

## 🚀 Getting Started

1. **Clone the repository**
```bash
   git clone https://github.com/yourusername/my-game.git
```
2. **Open in Visual Studio** — open `MyGame.sln`
3. **Build & Run** — press `F5` or click **Start**
